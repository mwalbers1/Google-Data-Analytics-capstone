### Data Set

**<a href="https://www.kaggle.com/datasets/arashnic/fitbit" target="_blank">FitBit Fitness Tracker Data</a>** 

This data set contains personal fitness tracker data from thirty fitbit users. Thirty eligible Fitbit users consented to the submission of personal tracker data, including minute-level output for physical activity, heart rate, and sleep monitoring. It includes information about daily activity, steps, and heart rate that can be used to explore usersâ€™ habits.
